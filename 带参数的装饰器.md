# 问题： 结合「变量作用域 + 闭包 + 装饰器」知识点，实现一个带参数的装饰器：
装饰器add_log可接收一个参数level（日志级别，如 "INFO"/"ERROR"）；
装饰器功能：在被装饰函数执行前打印「[级别] 函数 XXX 开始执行」，执行后打印「[级别] 函数 XXX 执行结束」；
用该装饰器装饰两个不同函数（如add_num求和、print_text打印文本），分别传入不同level参数；
调用两个函数，验证装饰器功能。


# 思路
带参数的装饰器需要三层函数嵌套：外层接收装饰器参数，中层接收原函数，内层实现功能；
核心逻辑：利用闭包保存装饰器参数（level）和原函数信息；

# 代码
```
# 带参数的装饰器：3层嵌套结构
def add_log(level):  # 第1层：接收装饰器参数（日志级别）
    def decorator(fn):  # 第2层：接收原函数
        def inner(*args, **kwargs):  # 第3层：实现装饰器功能
            # 获取被装饰函数的真实名称，替代硬编码的XXX
            func_name = fn.__name__
            # 执行前打印日志
            print(f'[{level}] 函数{func_name}开始执行，参数：{args} {kwargs}')
            # 执行原函数并接收返回值（保留原函数功能）
            result = fn(*args, **kwargs)
            # 执行后打印日志
            print(f'[{level}] 函数{func_name}执行结束,返回值{result}')
            # 返回原函数的返回值（避免原函数返回值丢失）
            return result
        return inner  # 第2层返回内函数（替代原函数）
    return decorator  # 第1层返回装饰器函数

# 装饰器使用：@装饰器名(参数)
@add_log("INFO")  # 为add_num添加INFO级别的日志装饰器
def add_num(a, b):
    return a + b

@add_log("ERROR")  # 为print_text添加ERROR级别的日志装饰器
def print_text(text):
    print(f"打印文本：{text}")

# 测试调用
print("add_num执行结果：", add_num(10, 20))
print("-" * 30)
print_text("Python带参数装饰器学习")
```
