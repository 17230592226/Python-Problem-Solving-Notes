#题目
1. 父类Goods（商品基础类）  
封装：私有属性__goods_id（商品编号）、__price（商品价格）；公共属性name（商品名称）、stock（库存）。  
@property：为__goods_id提供仅访问接口（不可修改）；为__price提供访问 + 修改接口，修改时做价格校验：价格必须≥0，否则赋值为 0 并打印提示"价格不能为负，已重置为0"。  
魔术方法：重写__str__，返回格式："商品编号：{__goods_id}，商品名称：{name}，价格：{__price}元，库存：{stock}件"。  
类方法：create_default，创建默认商品对象（编号：DEFAULT001，名称：默认商品，价格：9.9，库存：100），返回该对象。  
静态方法：check_stock，接收stock参数，判断库存是否＞0：是则返回True，否则返回False。  
2. 子类Electronics（电子产品类，继承 Goods）  
新增封装：私有属性__brand（品牌），为其提供访问 + 修改接口。  
方法重写（多态）：重写__str__，返回格式："商品编号：{__goods_id}，商品名称：{name}，品牌：{__brand}，价格：{__price}元，库存：{stock}件"。  
新增实例方法：show_warranty，打印"{name}（{__brand}）质保1年"。  
3. 子类Clothes（服饰类，继承 Goods）  
新增封装：私有属性__size（尺码，如 M/L/XL），为其提供访问 + 修改接口。  
方法重写（多态）：重写__str__，返回格式："商品编号：{__goods_id}，商品名称：{name}，尺码：{__size}，价格：{__price}元，库存：{stock}件"。  
新增静态方法：check_size，接收size参数，判断是否为["S","M","L","XL","XXL"]中的值：是则返回True，否则返回False。

class Goods:
    def __init__(self, goods_id, price,name,stock):
        self.__goods_id = goods_id
        self.__price = price
        self.name = name
        self.stock = stock

    @property
    def goods_id(self):
        return self.__goods_id

    @property
    def price(self):
        return self.__price

    @price.setter
    def price(self, new_price):
        if new_price >=0:
            self.__price = new_price
        else:
            self.__price = 0
            print('价格不能为负，已重置为0')

    def __str__(self):
        return f'商品编号：{self.__goods_id}，商品名称：{self.name}，价格：{self.__price}元，库存：{self.stock}件'

    @classmethod
    def create_default(cls):
        return cls('DEFAULT001',9.9,'默认商品',100)

   #静态方法：check_stock，接收stock参数，判断库存是否＞0：是则返回True，否则返回False
    @staticmethod
    def create_stock(stock):
        if stock > 0:
            return True
        else:
            return False

class Electronics(Goods):

    def brand(self):
        return self.__brand

    def brand(self, new_brand):

# 新增静态方法：check_size，接收size参数，判断是否为["S","M","L","XL","XXL"]中的值：是则返回True，否则返回False。

class Clothes(Goods):
    def __init__(self, goods_id, price, name, stock,size):
        super().__init__(goods_id, price, name, stock)
        self.__size = size

    @property
    def size(self):
        return self.__size

    @size.setter
    def size(self, new_size):
        self.__size = new_size
    def __str__(self):
        return f'商品编号：{self.goods_id}，商品名称：{self.name}，尺码：{self.__size}，价格：{self.price}元，库存：{self.stock}'
    @staticmethod
    def show_warranty(size):
        if size in ["S","M","L","XL","XXL"]:
            return True
        else:
            return False
