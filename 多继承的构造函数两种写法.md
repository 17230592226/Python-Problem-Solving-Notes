# 问题：多继承的构造函数两种写法
父类 1Father：构造方法初始化height（身高），方法show_height打印"身高：{height}cm"；  
父类 2Mother：构造方法初始化weight（体重），方法show_weight打印"体重：{weight}kg"；  
子类Child：继承Father和Mother，构造方法初始化height、weight、name（姓名），需分别调用两个父类的构造方法；新增方法show_all，打印"姓名：{name}，身高：{height}cm，体重：  {weight}kg"；
创建Child对象child = Child(175, 65, "小明")，依次调用show_height、show_weight、show_all方法。  

# 思路：
print(Child.__mro__)  
(<class '__main__.Child'>, <class '__main__.Father'>, <class '__main__.Mother'>, <class 'object'>)  
这个顺序的含义是：调用方法时，Python 会按「Child → Father → Mother → object」的顺序查找，找到第一个匹配的方法就执行。  

# 代码
```
class Father:
    def __init__(self, height):
        self.height = height

    def show_height(self):
        print(f'身高：{self.height}cm')


class Mother:
    def __init__(self, weight):
        self.weight = weight

    def show_weight(self):
        print(f'体重：{self.weight}kg')

class Child(Father, Mother): #Child -> Father ->Mother
    def __init__(self, height,weight,name):
        # Father.__init__(self, height)
        # Mother.__init__(self, weight)
        super().__init__(height)  # 隐式调用 super(Child,self).__init__(height)
        super(Father, self).__init__(weight)  #super(Father,self).__init__(weight)
        self.name = name

    def show_all(self):
        print(f"姓名：{self.name}，身高：{self.height}cm，体重：{self.weight}kg")

child = Child(175, 65, "小明")
child.show_height()
child.show_weight()
child.show_all()
print(Child.__mro__)
```
