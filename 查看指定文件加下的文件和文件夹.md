# 题目
编写代码，遍历当前目录下的所有文件和文件夹；  
&nbsp;&nbsp;对遍历到的每个路径，分别判断：  
&nbsp;&nbsp;如果是文件夹，打印：[文件夹] 路径名；  
&nbsp;&nbsp;如果是文件，打印：[文件] 路径名 - 大小：XXX字节；  
&nbsp;&nbsp;要求使用 os.listdir() 和 os.path 相关函数实现。  

# 代码
```
import os

# 获取当前工作目录
current_dir = os.getcwd()
# 遍历当前目录下的所有内容
for name in os.listdir(current_dir):
    # 拼接完整路径（关键：避免路径错误）
    full_path = os.path.join(current_dir, name)
    # 判断是否为文件夹
    if os.path.isdir(full_path):
        print(f"[文件夹] {full_path}")
    # 判断是否为文件
    elif os.path.isfile(full_path):
        # 获取文件大小
        file_size = os.path.getsize(full_path)
        print(f"[文件] {full_path} - 大小：{file_size}字节")
```
